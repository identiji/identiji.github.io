Q: Why was Beta launched as a web app? I thought you were going to make a browser extension first.


A: In a previous survey, most of you expressed interest in using Identiji on a mobile browser (rather than a just desktop one).


By making Identiji available as a web app, we are able to let a larger number of users access the Identiji service right away, without having to install extra plugins or extensions â€” which usually only end up working on one platform but not the other, and can therefore be an inconvenience.


We understand that having to open Identiji.com everytime you need to enter a password may not be ideal for everyone, so we may still consider releasing a browser extension in the future.





Q: I am a web developer. Can I integrate Identiji onto my site?


A: Quite possibly in the near future.


Since Day 1, we have been contemplating releasing a developers' tool version of Identiji. That way more third-party websites out there can have native support of emoji passwords that are web-friendly.


Now that the web app beta is in a stable stage, we might start working on the dev tool soon.


And when that happens, we will most likely open source the code, too.
